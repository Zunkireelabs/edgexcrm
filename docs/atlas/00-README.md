## What EdgeX is

A **multi-tenant, multi-industry lead-generation CRM** (Zunkiree Labs / "EdgeX"). One Next.js 16 App Router app (React 19 + TypeScript) backed by **Supabase** (Postgres + Auth + Storage), with Row-Level Security enforcing tenant isolation. Each tenant belongs to exactly one **industry**, and an **industry-module registry** gates which features that tenant sees. The two built-out industries are **education-consultancy** and **it-agency**; the other six are placeholder manifests. Deployed as a Docker standalone image behind Traefik across three tiers (local → staging → production), each with its own database.

### Domains (Industries)

Each industry is its own product. These are **dual-audience** — a **Business Logic** tab (plain language for PMs, product, QC, and clients, with UX journeys and AI/automation opportunities) and an **Engineering Logic** tab (system design, DB schema, component relations for devs).
